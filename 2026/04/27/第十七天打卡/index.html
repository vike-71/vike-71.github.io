问题描述

给定一个字符串 s，你需要将其中的单词顺序反转。单词由非空格字符组成，单词之间由至少一个空格分隔。

返回一个字符串，其中单词顺序反转，并用单个空格连接。注意，输入字符串可能包含前导或尾随空格，或单词之间有多个空格，但输出字符串不应包含前导或尾随空格，且单词之间只有一个空格。

示例：

  • 输入: s = “the sky is blue”
  • 输出: “blue is sky the”

解题思路

解决这个问题，我们采用原地修改字符串的方法，通过两次反转来实现单词顺序的反转。

首先，将整个字符串反转。此时，单词的顺序已经反转，但每个单词的字符顺序也被反转了。

接下来，我们需要修正每个单词的字符顺序，并处理空格。

使用两个指针：快指针 fast 用于遍历原字符串，慢指针 slow 用于构建新字符串。

  • 跳过前导空格：在开始处理之前，使用 fast 指针跳过字符串开头的连续空格，确保我们从第一个非空格字符开始。这是因为输入字符串可能有前导空格，我们不需要将它们复制到结果中。具体来说，使用一个内层循环 while (fast < c.length && c[fast] == ' ') fast++;，如果 fast 到达字符串末尾，则表示整个字符串都是空格，直接跳出外层循环。

  • 当遇到非空格字符时，开始一个单词：记录单词起始位置 index1 = slow，然后将单词字符复制到 slow 位置，同时移动 slow 和 fast。

  • 复制完单词后，反转该单词的字符（从 index1 到 slow-1）。

  • 如果还有空间，添加一个空格到 slow 位置。

重复此过程，直到遍历完整个字符串。

  • 去除末尾的多余空格：在所有单词处理完毕后，由于我们可能在最后一个单词后添加了一个空格，但如果没有更多单词，这个空格就是多余的。因此，使用一个循环 while (slow > 0 && c[slow-1] == ' ') slow--;，从 slow-1 开始向前检查，如果遇到空格，就减少 slow，直到遇到非空格字符或 slow 为 0。这样确保结果字符串没有尾随空格，并且不会删除单词中的空格。

最后，返回从 0 到 slow 的子字符串。

时间复杂度是 O(n)，其中 n 是字符串的长度。具体来说：

  • 整体反转字符串需要 O(n) 时间，因为需要遍历整个字符串进行交换。
  • 遍历字符串并处理单词的过程中，快指针 fast 和慢指针 slow 各遍历字符串一次，时间为 O(n)，在此过程中还包括跳过空格和复制字符的操作。
  • 反转每个单词的时间总和也是 O(n)，因为每个字符在整个过程中最多被反转两次（一次整体反转，一次单词反转），但实际上单词反转只针对单词字符。

因此，总时间复杂度为 O(n)，这意味着算法的运行时间与输入字符串长度成线性关系，非常适合处理长字符串。

空间复杂度是 O(n)，因为我们将输入字符串转换为字符数组以进行原地修改。这个字符数组的大小与输入字符串长度成正比。除了这个数组，我们只使用了常数个额外变量，因此总空间复杂度为 O(n)。
