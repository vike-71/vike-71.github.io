LeetCode题目

  • 题目要求 : 给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。
  • 变量定义 :
  • start : 表示当前层的起始位置
  • cnt : 表示当前已经填充的元素个数
  • 解题思路 :
  • 因为是一个n x n的正方形矩阵,所以第一圈的起始位置从0开始,每圈的起始位置都比上一圈大1,所以start初始值为0,每圈结束后start加1 int start= 0;
  • 因为数是从1开始的,所以cnt初始值为1 int cnt=1;
  • 因为是一个n x n的正方形矩阵,所以会有n/2圈,所以当start小于n/2时,说明还没有填充完,所以循环条件为 while(start<(n/2))
  • 每圈的填充顺序是从左到右,从上到下,从右到左,从下到上,且每填充一个元素cnt加1所以每层的填充顺序为:
    • 从左到右填充: for (int i=start;i<n-start-1;i++){ matrix[start][i]=cnt++; }
    • 从上到下填充: for (int i=start;i<n-start-1;i++){ matrix[i][n-start-1]=cnt++; }
    • 从右到左填充: for (int i=n-start-1;i>start;i--){ matrix[n-start-1][i]=cnt++; }
    • 从下到上填充: for (int i=n-start-1;i>start;i--){ matrix[i][start]=cnt++; }
    • 因为当n是奇数时,最后会比n为偶数多一个元素没有填充,所以将该元素进行填充 if ((n%2)==1) { matrix[start][start]=cnt; }
