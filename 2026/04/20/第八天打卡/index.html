LeetCode题目

  • 题目要求 : 给你两个按 非递减顺序 排列的整数数组nums1 和 nums2，另有两个整数 m 和 n ，分别表示 nums1 和 nums2 中的元素数目。请你 合并 nums2 到 nums1 中，使合并后的数组同样按 非递减顺序 排列。
  • 解题思路 :
    • 因为给你两个按 非递减顺序 排列的整数数组nums1 和 nums2,相当于归并排序中已经排好序的左部分和右部分,所以我先想到的是用归并排序中merge的过程来将两个数组合并

使用归并排序中merge的过程来将两个数组合并

  • 变量定义 :
  • index1 : 表示nums1数组中有值的索引位置
  • index2 : 表示nums2数组的索引位置
  • i : 表示辅助数组nums3的索引位置
  • nums3 : 表示辅助数组,用来存储合并后的结果
  • 思路 :
  • 因为nums1数组中有值的索引位置为0到m-1,所以index1初始值为0,当index1小于m时说明nums1数组中还有值
  • 因为nums2数组的索引位置为0到n-1,所以index2初始值为0,当index2小于n时说明nums2数组中还有值,所以循环条件为 while (index1 < m && index2 < n)
  • 因为nums1数组和nums2数组都是按 非递减顺序 排列的,所以每次比较nums1数组中index1位置的值和nums2数组中index2位置的值,将较小的值添加到辅助数组nums3中,并且将对应的索引位置加1,所以 nums3[i++] = nums1[index1] < nums2[index2] ? nums1[index1++] : nums2[index2++];
  • 当右部分索引越界后,将左部分中剩余的元素全部添加到辅助数组nums3中while (index1 < m) { nums3[i++] = nums1[index1++]; }
  • 当左部分索引越界后,将右部分中剩余的元素全部添加到辅助数组nums3中while (index2 < n) { nums3[i++] = nums2[index2++]; }
  • 将辅助数组nums3中的元素全部添加到nums1数组中for (i = 0; i < m + n; i++) { nums1[i] = nums3[i]; }

使用视频中的解法

  • 变量定义 :
  • index1 : 表示nums1数组中有值的从尾部开始的索引位置
  • index2 : 表示nums2数组的从尾部开始的索引位置
  • i : 表示nums1数组的从尾部开始的索引位置
  • 思路 :
  • 因为nums1数组中有值的索引位置为0到m-1,所以index1初始值为m-1,当index1大于等于0时说明nums1数组中还有值
  • 因为nums2数组的索引位置为0到n-1,所以index2初始值为n-1,当index2大于等于0时说明nums2数组中还有值,所以循环条件为 while (index1 >= 0 && index2 >= 0)
  • 因为nums1数组和nums2数组都是按 非递减顺序 排列的,所以每次比较nums1数组中index1位置的值和nums2数组中index2位置的值,将较大的值添加到nums1数组中的i位置,并且将对应的索引位置减1,所以 nums1[i--] = nums1[index1] < nums2[index2] ? nums2[index2--] : nums1[index1--];
  • 当左部分索引越界后,将右部分中剩余的元素全部添加到nums1数组中while (index2 >= 0) { nums1[i--] = nums2[index2--]; }
  • 因为当右部分索引越界后,左部分中剩余的元素已经在nums1数组中了,所以不需要再进行操作
