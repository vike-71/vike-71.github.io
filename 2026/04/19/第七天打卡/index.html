LeetCode题目

  • 题目要求 : 给定一个链表的头节点 head，返回链表开始入环的第一个节点。如果链表无环，则返回null。
  • 变量定义 :
  • fast: 表示快指针
  • slow: 表示慢指针
  • index1: 表示从头节点开始的指针
  • index2: 表示从相遇节点开始的指针
  • 解题思路 :
  • 因为需要判断链表是否有环,所以需要一个快指针和一个慢指针,快指针每次移动两步,慢指针每次移动一步,如果快指针和慢指针相遇了,说明链表有环,否则链表无环,且快指针比慢指针快(每次移动两步),只需判断快指针和快指针下一个位置是否为null所以循环条件为 while (fast != null && fast.next != null)
    • 如果快指针和慢指针相遇了,说明链表有环,所以需要一个指针从头节点开始,一个指针从相遇节点开始,每次移动一步,直到两个指针相遇了,说明找到了链表开始入环的第一个节点,所以循环条件为 while (index1 != index2)
      • 如果快指针和慢指针没有相遇了,说明链表无环,所以返回null return null;
      • 如果快指针和慢指针相遇了,说明链表有环,所以返回相遇节点 return index1;
