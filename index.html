LeetCode题目

  • 题目要求 : 给你一个链表的头节点 head 和一个整数 val ，请你删除链表中所有满足 Node.val == val 的节点，并返回 新的头节点 。
  • 思路 : 使用虚拟头节点
  • 变量定义 :
  • dummyHead : 表示虚拟头节点
  • cur : 表示当前节点
  • 解题思路 :
  • 因为要删除链表中所有满足 Node.val == val 的节点,所以需要一个虚拟头节点来处理头节点被删除的情况,所以dummyHead初始值为一个新的节点,且该节点的next指向head ListNode dummyHead = new ListNode(-1,head);
  • 因为需要遍历链表来删除满足条件的节点,所以cur初始值为虚拟头节点 ListNode cur = dummyHead;
  • 当cur的下一个节点不为空时,说明还有节点需要判断,所以循环条件为 while(cur.next!=null)
    • 如果cur的下一个节点的值等于val,说明需要删除该节点,所以将cur的下一个节点指向下一个节点的下一个节点 cur.next = cur.next.next;
    • 如果cur的下一个节点的值不等于val,说明不需要删除该节点,所以将cur指向下一个节点 cur = cur.next;
    • 最后返回虚拟头节点的下一个节点,即新的头节点 return dummyHead.next;

LeetCode题目

  • 题目要求 : 给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。
  • 变量定义 :
  • start : 表示当前层的起始位置
  • cnt : 表示当前已经填充的元素个数
  • 解题思路 :
  • 因为是一个n x n的正方形矩阵,所以第一圈的起始位置从0开始,每圈的起始位置都比上一圈大1,所以start初始值为0,每圈结束后start加1 int start= 0;
  • 因为数是从1开始的,所以cnt初始值为1 int cnt=1;
  • 因为是一个n x n的正方形矩阵,所以会有n/2圈,所以当start小于n/2时,说明还没有填充完,所以循环条件为 while(start<(n/2))
  • 每圈的填充顺序是从左到右,从上到下,从右到左,从下到上,且每填充一个元素cnt加1所以每层的填充顺序为:
    • 从左到右填充: for (int i=start;i<n-start-1;i++){ matrix[start][i]=cnt++; }
    • 从上到下填充: for (int i=start;i<n-start-1;i++){ matrix[i][n-start-1]=cnt++; }
    • 从右到左填充: for (int i=n-start-1;i>start;i--){ matrix[n-start-1][i]=cnt++; }
    • 从下到上填充: for (int i=n-start-1;i>start;i--){ matrix[i][start]=cnt++; }
    • 因为当n是奇数时,最后会比n为偶数多一个元素没有填充,所以将该元素进行填充 if ((n%2)==1) { matrix[start][start]=cnt; }

LeetCode题目

  • 题目要求 : 给定一个含有n个正整数的数组和一个正整数 target ,找出该数组中满足其总和大于等于 target 的长度最小的 子数组[numsl, numsl+1, …, numsr-1, numsr] ，并返回其长度。如果不存在符合条件的子数组，返回 0 。
  • 题目分析 :
  • 数组的范围和累加和具有单调性关系: 范围越大,累加和越大;范围越小,累加和越小.
  • 所以可以使用滑动窗口来解决这个问题.

使用滑动窗口解题(求子数组在每个位置 结尾 情况下的答案)

  • 变量定义 :
  • index : 表示满足条件的子数组的最小长度
  • l : 表示滑动窗口的左边界
  • r : 表示滑动窗口的右边界
  • sum : 表示滑动窗口内元素的总和
  • 思路 :
  • index 初始赋值为整数的最大值,表示最开始没有找到满足条件的子数组 int index = Integer.MAX_VALUE;
  • sum 初始赋值为0,表示滑动窗口最开始内元素的总和为0 int sum = 0;
  • sum+=nums[r];和for循环中的r++; 表示每次循环都将右边界向右移动一位,并将新加入滑动窗口的元素值加到sum上
  • sum-=nums[l++];和while循环的循环条件sum-nums[l]>=target 表示当滑动窗口内元素的总和减去左边界的元素值仍然大于等于target时,就将左边界向右移动一位,并将移出滑动窗口的元素值从sum中减去,用于求出结尾以r位置为右边界的满足条件的子数组的最小长度的情况
  • 如果 ‘sum>=target’ ,说明当前滑动窗口内元素的总和已经满足条件了,就更新index的值为当前滑动窗口的长度和之前index值的较小值 index = Math.min(index, r - l + 1);
  • 最后返回index的值,如果index仍然是整数的最大值,说明没有找到满足条件的子数组,就返回0,否则返回index的值 return index==Integer.MAX_VALUE?0:index;

LeetCode题目链接

LeetCode题目分析

  • 题目要求 : 给定一个n个元素有序的（升序）整型数组nums 和一个目标值target ，写一个函数搜索nums中的 target，如果target存在返回下标，否则返回 -1.
  • 题目分析 : 因为是有序数组找一个数,所以使用二分查找
  • 变量定义 :
  1. l: 表示左边界
  2. r: 表示右边界
  3. m: 表示中间位置
  4. index: 表示目标值的索引

使用二分查找解题(左闭右闭)

思路:

  • 因为是左闭右闭且如果target不存在返回-1,所以 int index=-1,l=0,right-nums.length-1;
  • 因为是左边右闭,所以循环条件为 l<=r
  • 计算中间位置:m=l+((r-l)>>1);(防止整数溢出)
  • 如果nums[m] == target,另index 等于当前中间位置的索引值 所以 index = m,因为找到了目标值索引,所以退出循环
  • 如果nums[m] < target,说明目标值在中间位置的右侧,得往右边找, 所以 l=m+1
  • 如果nums[m] > target,说明目标在中间位置的左侧,得往右边找且为左闭右闭, 所以 r=m-1

使用二分查找解题(左闭右开)

思路:

  • 因为是左闭右开且如果target不存在返回-1,所以 int index=-1,l=0,right-nums.length;
  • 因为是左边右开,所以循环条件为 l<r
  • 计算中间位置:m=l+((r-l)>>1);(防止整数溢出)
  • 如果nums[m] == target,另index 等于当前中间位置的索引值 所以 index = m,因为找到了目标值索引,所以退出循环
  • 如果nums[m] < target,说明目标值在中间位置的右侧,得往右边找, 所以 l=m+1
  • 如果nums[m] > target,说明目标在中间位置的左侧,得往右边找且为左闭右开, 所以 r=m
