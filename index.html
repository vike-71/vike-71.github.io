LeetCode题目分析

  • 题目要求 : 给定一个n个元素有序的（升序）整型数组nums 和一个目标值target ，写一个函数搜索nums中的 target，如果target存在返回下标，否则返回 -1.
  • 题目分析 : 因为是有序数组找一个数,所以使用二分查找
  • 变量定义 :
  1. l: 表示左边界
  2. r: 表示右边界
  3. m: 表示中间位置
  4. index: 表示目标值的索引

使用二分查找解题(左闭右闭)

思路:

  • 因为是左闭右闭且如果target不存在返回-1,所以 int index=-1,l=0,right-nums.length-1;
  • 因为是左边右闭,所以循环条件为 l<=r
  • 计算中间位置:m=l+((r-l)>>1);(防止整数溢出)
  • 如果nums[m] == target,另index 等于当前中间位置的索引值 所以 index = m,因为找到了目标值索引,所以退出循环
  • 如果nums[m] < target,说明目标值在中间位置的右侧,得往右边找, 所以 l=m+1
  • 如果nums[m] > target,说明目标在中间位置的左侧,得往右边找且为左闭右闭, 所以 r=m-1

使用二分查找解题(左闭右开)

思路:

  • 因为是左闭右开且如果target不存在返回-1,所以 int index=-1,l=0,right-nums.length;
  • 因为是左边右开,所以循环条件为 l<r
  • 计算中间位置:m=l+((r-l)>>1);(防止整数溢出)
  • 如果nums[m] == target,另index 等于当前中间位置的索引值 所以 index = m,因为找到了目标值索引,所以退出循环
  • 如果nums[m] < target,说明目标值在中间位置的右侧,得往右边找, 所以 l=m+1
  • 如果nums[m] > target,说明目标在中间位置的左侧,得往右边找且为左闭右开, 所以 r=m
